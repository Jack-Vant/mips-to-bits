    Considerations

    Drag and drop is available

    Adding does not update old file(s), remove the old file then add the new one

    Labels may contain any letters, numbers and "_" but must not start with a number

    Floating point and pseudo instructions are not available

    There is currently no plan to add them (fork 'em)

    2020 JavaScript required
